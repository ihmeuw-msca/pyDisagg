===================
Installing template
===================

Python version
--------------

The package :code:`template` is written in Python
and requires Python 3.10 or later.

Install template
----------------

Regmod package is distributed at
`PyPI <https://pypi.org/project/template/>`_.
To install the package:

.. code::

   pip install template

For developers
--------------

For developers, you can clone the repository and install the package in the
development mode.

.. code::

    git clone https://github.com/ihmeuw-msca/template.git
    cd template
    pip install -e ".[test,docs]"
